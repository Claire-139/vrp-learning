# vrp-learning

学习OR-Tools解决车辆路径问题（VRP）

## 项目结构
- vrp_demo.py：VRP  
- cvrp_demo.py：带载重限制的VRP
- vrptw_demo.py：带时间窗的VRP  
- multi_depot_vrp.py：多起点VRP + 可视化

## 环境
- Python 3.x
- ortools
- matplotlib

## 运行方式
python vrp_demo.py

## 问题描述
本项目使用Google OR-Tools解决以下VRP变体：
- VRP：基础车辆路径问题
- CVRP：带载重限制的VRP
- VRPTW：带时间窗的VRP  
- Multi-Depot VRP：多仓库VRP，支持路径可视化
